An undergraduate group project to simulate computerized tomography using Matlab.

Useful Links

CTSim A very good CT simulation software. The only drawback is that it does not provide instruction for Direct Fourier Reconstruction.

Aims

  • To be able to simulate CT reconstruction using Central Slice Theorem
  • To study a number of factors on image reconstruction:
    1. Number of sensors
    2. Number of projection slices
    3. Scan angle range
    4. Noise
    5. Sensor damage

Simulation Program

Program summary

  1. Generate a phantom
  2. Apply Radon Transform from theta=0deg to theta=180deg
  3. Do 1D Fourier Transform to each angle theta
  4. Interpolate data to change from polar coordinates to rectangular coordinates
  5. Apply inverse 2D Fourier Transforms.
